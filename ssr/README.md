# SSR Hosting

## Set-up

```sh
$ npm install
$ npm run build
```

This sample app is created by [Next.js](https://github.com/zeit/next.js/)

## Serve

* [AWS](use-aws.md)
  * Example: https://d35olaev50y0fr.cloudfront.net

* [Firebase](use-firebase.md)
